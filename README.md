 MindEase — Student Wellbeing App

🏆 Dunes Interschool Hackathon Prototype Pioneer Winner!

MindEase is a student wellness desktop app that reduces academic stress through mood tracking, AI planning, stress assessments, and a built in AI chatbot for personal support all in one clean, dark-themed interface.

Features

🏠 Home DashboardSee your App Streak, Mood Streak, last Survey Result, and AI Reports at a glance
😊 Mood & Energy TrackerLog how you're feeling daily and track patterns over time
📋 SurveyAnswer quick wellness surveys to measure your stress levels
🧠  AssessmentGet a deeper evaluation of your mental wellbeing
📅 AI PlannerAutomated timetable scheduling powered by Gemini AI
😌 RelaxBuilt-in relaxation tools to help you unwind
🤖 AI HelperA Gemini-powered chatbot available anytime for support and advice
📊 ReportView AI-generated summaries and insights about your wellbeing

🛠️ Built With

Python — Core programming language
CustomTkinter — Modern dark-themed desktop UI
Tkinter — GUI base framework
Gemini API — Powers the AI Planner and AI Helper chatbot
Requests — Handles API communication
Threading — Smooth background processing
